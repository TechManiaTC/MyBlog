### *如何使用本代码*：
### 在Linux环境下打开终端，键入以下命令：
1. git clone https://github.com/TechManiaTC/MyBlog.git
2. cd MyBlog
3. virtualenv --python=python3 venv
4. source venv/bin/activate
5. pip install -r requirements.txt
6. python manage.py runserver
